# 🌦️ WeatherCast:
### Real-Time Weather Application

🔗 **Live Demo:** https://weathercast-se1i.onrender.com  

---

## 📌 Overview
WeatherCast is a web-based application that provides real-time weather information for any city. Users can search for a location and instantly view current weather details such as temperature, humidity, wind speed, and weather conditions.

This project focuses on a clean user interface, responsive design, and seamless integration with a weather API.

---

## 🚀 Features
- 🌍 Search weather by city name  
- 🌡️ Displays temperature, humidity, and wind speed  
- ☁️ Shows weather conditions (Clear, Clouds, Rain, etc.)  
- 📱 Fully responsive design (mobile + desktop)  
- ⚡ Fast and user-friendly interface  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **API:** OpenWeatherMap (or any weather API used)  
- **Deployment:** Render  

---

## ⚙️ How It Works
1. User enters a city name in the search bar  
2. Application sends a request to the weather API  
3. API returns real-time weather data  
4. Data is dynamically displayed on the UI
